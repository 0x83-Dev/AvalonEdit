Namespaces ► ICSharpCode.AvalonEdit.Folding ► FoldingManager ► UpdateFoldings(IEnumerable<(Of <<'(NewFolding>)>>), Int32)
Updates the foldings in this FoldingManager using the given new foldings. This method will try to detect which new foldings correspond to which existing foldings; and will keep the state (IsFolded) for existing foldings.

firstErrorOffset (Int32)
The first position of a parse error. Existing foldings starting after this offset will be kept even if they don't appear in newFoldings. Use -1 for this parameter if there were no parse errors.
