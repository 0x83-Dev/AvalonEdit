StartListening(Object)
When overridden in a derived class, starts listening for the event being managed. After StartListening(Object) is first called, the manager should be in the state of calling DeliverEvent(Object, EventArgs) or DeliverEventToList(Object, EventArgs, WeakEventManager..::..ListenerList) whenever the relevant event from the provided source is handled.
(Overrides WeakEventManager.StartListening(Object).)
